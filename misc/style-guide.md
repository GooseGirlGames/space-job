# Styleguide

## Naming conventions

- Private members like: *m_FooBar*
- universaly CammelCase
- mehodes Big 

## Directory structure

- Blender files in the `Assets/Models/blend` directory, exported models in the models directory
- GIMP files in `Assets/Textures/gimpfiles`, textures in `Assets/Textures`
- all Prefabs need to be created in the `Assets/Prefabs` directory
